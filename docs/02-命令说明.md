## 基础命令

### mesh init

初始化配置，设置同步空间路径。

```bash
mesh init --sync-space /path/to/sync
```

**参数**:
- `--sync-space`: 同步空间路径（必填）

**示例**:
```bash
mesh init --sync-space /Users/xiaolin/Downloads/同步空间/.mesh
```

---

### mesh collect

采集数据到本地数据库。

```bash
mesh collect --source "claude" --content "内容" --tag "标签"
```

**参数**:
- `--source`: 数据来源（必填）
  - 可选值：`claude`, `chatgpt`, `cursor`, `clipboard`, `file`, `stdin`, 自定义
- `--content`: 数据内容（必填）
- `--tag`: 标签（可选，逗号分隔）
- `--summary`: 摘要（可选）

**示例**:
```bash
# 采集决策
mesh collect --source "claude" --content "决策：使用 Go 开发" --tag "技术,决策"

# 从剪贴板
mesh collect --source "clipboard" --content "$(pbpaste)" --tag "笔记"

# 带摘要
mesh collect \
  --source "claude" \
  --content "长内容..." \
  --summary "关于定价的讨论" \
  --tag "产品,定价"
```

---

### mesh query

查询数据。

```bash
mesh query "关键词" [选项]
```

**参数**:
- `关键词`: 搜索词（必填）
- `--source`: 按来源过滤
- `--tag`: 按标签过滤
- `--limit`: 限制结果数（默认 10）
- `--format`: 输出格式
  - `text` (默认) - 文本格式
  - `json` - JSON 格式

**示例**:
```bash
# 关键词搜索
mesh query "定价策略"

# 按来源过滤
mesh query --source "claude"

# 按标签过滤
mesh query --tag "产品"

# 组合查询
mesh query --source "claude" --tag "决策" --limit 5

# JSON 格式输出
mesh query "定价" --format json
```

**输出示例**:
```
找到 3 条相关认知

1. [claude, 2026-03-10 14:30]
   决策：采用分层定价，基础版免费，专业版 99 元/月
   标签: 产品,定价

2. [chatgpt, 2026-03-09 10:15]
   竞品分析：主要竞品定价为 79-129 元/月
   标签: 产品,定价

3. [cursor, 2026-03-08 16:45]
   技术实现：需要支持支付宝和微信支付
   标签: 产品,定价
```

### mesh list

列出数据。

```bash
mesh list [选项]
```

**参数**:
- `--source`: 按来源过滤
- `--limit`: 限制结果数（默认 10）

**示例**:
```bash
# 列出最近 10 条
mesh list

# 列出最近 20 条
mesh list --limit 20

# 列出 claude 来源的
mesh list --source "claude"
```

**输出示例**:
```
最近 10 条认知

ID        | Source   | Created At   | Tags
--------------------------------------------------
xxx       | claude   | 2026-03-10   | 技术,决策
xxx       | chatgpt  | 2026-03-09   | 产品,定价
...
```

## 同步命令

### mesh push

推送数据到同步空间。

```bash
mesh push
```

**说明**: 复制本地数据库到同步空间

---

### mesh pull

从同步空间拉取数据。

```bash
mesh pull
```

**说明**: 复制同步空间数据库到本地

---

### mesh sync

双向同步。

```bash
mesh sync
```

**说明**: 先 pull，再 push

**输出**:
```
✅ 同步完成
  拉取: 5 条新记录
  推送: 3 条新记录
```

## 导入导出

### mesh export

导出数据为 Markdown。

```bash
mesh export [选项]
```

**参数**:
- `--output`: 输出文件（可选，默认 stdout）

**示例**:
```bash
# 导出到 stdout
mesh export

# 导出到文件
mesh export --output memory.md

# 导出特定来源
mesh export --source "claude" --output claude.md
```

---

### mesh import

从 Markdown 导入数据。

```bash
mesh import <文件>
```

**参数**:
- `<文件>`: Markdown 文件路径（必填）

**示例**:
```bash
# 导入现有 memory
mesh import /path/to/memory.md
```

**支持的 Markdown 格式**:
```markdown
## 2026-03-10

### 决策：使用 Go 开发 Mesh
标签: 技术,决策
内容: xxx

...
```

## Agent 命令

### mesh agent register

注册 Agent。

```bash
mesh agent register \
  --id "agent_id" \
  --name "Agent Name" \
  --type "type" \
  --device "device"
```

**参数**:
- `--id`: Agent ID（必填）
- `--name`: Agent 名称（必填）
- `--type`: Agent 类型（必填）
- `--device`: 设备标识（必填）
- `--metadata`: 元数据（JSON 格式，可选）

**示例**:
```bash
mesh agent register \
  --id "cursor_main_shawn" \
  --name "Cursor Main" \
  --type "cursor" \
  --device "shawn" \
  --metadata '{"version":"1.0"}'
```

### mesh agent heartbeat

Agent 心跳。

```bash
mesh agent heartbeat --id "agent_id"
```

**说明**: Agent 定期调用此命令报告存活状态

## 上下文注入

### mesh inject

查询并格式化上下文，注入到新对话。

```bash
mesh inject --topic "主题" [选项]
```

**参数**:
- `--topic`: 查询主题（必填）
- `--max-tokens`: 最大 token 数（默认 2000）
- `--format`: 输出格式
  - `prompt` (默认) - 可直接粘贴的格式
  - `json` - JSON 格式

**示例**:
```bash
# 注入定价策略上下文
mesh inject --topic "定价策略"

# 限制 token 数
mesh inject --topic "定价策略" --max-tokens 1000
```

**输出示例**:
```
# 上下文信息

## 相关认知（2条）

1. [claude_main_xiaolin, 2026-03-10]
   决策：定价策略采用分层定价，基础版免费，专业版 99 元/月

2. [cursor_main_shawn, 2026-03-11]
   竞品分析：主要竞品定价为 79-129 元/月
```
