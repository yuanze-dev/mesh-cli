# Mesh 快速入门

> 5 分钟上手 Mesh CLI

---

## 一、安装

### 1.1 编译安装（开发者）

```bash
# 克隆项目
cd /Users/xiaolin/Downloads/同步空间/mesh

# 编译
go build -o mesh cmd/mesh/main.go

# 移动到 PATH
sudo mv mesh /usr/local/bin/
```

### 1.2 下载安装（用户）

```bash
# macOS Intel
curl -O https://releases.mesh.dev/mesh-darwin-amd64
chmod +x mesh-darwin-amd64
sudo mv mesh-darwin-amd64 /usr/local/bin/mesh

# macOS Apple Silicon
curl -O https://releases.mesh.dev/mesh-darwin-arm64
chmod +x mesh-darwin-arm64
sudo mv mesh-darwin-arm64 /usr/local/bin/mesh
```

## 二、初始化

### 2.1 首次使用

```bash
# 初始化（设置同步空间路径）
mesh init --sync-space /Users/xiaolin/Downloads/同步空间/.mesh

# 导出现有 memory（可选）
mesh import /Users/xiaolin/Downloads/同步空间/Claude code/memory/work/memory.md
```

### 2.2 验证安装

```bash
# 查看版本
mesh --version

# 查看帮助
mesh --help
```

## 三、基本使用

### 3.1 采集数据

```bash
# 采集一条认知
mesh collect --source "claude" --content "决策：使用 Go 开发 Mesh" --tag "技术,决策"

# 从剪贴板采集
mesh collect --source "clipboard" --content "$(pbpaste)" --tag "笔记"

# 从文件采集
mesh collect --source "file" --content "$(cat notes.txt)" --tag "文档"
```

### 3.2 查询数据

```bash
# 关键词查询
mesh query "定价策略"

# 按来源查询
mesh query --source "claude"

# 按标签查询
mesh query --tag "产品"

# 限制结果数
mesh query "最近" --limit 5
```

### 3.3 列出数据

```bash
# 列出最近 10 条
mesh list --limit 10

# 按来源列出
mesh list --source "claude"
```

### 3.4 同步

```bash
# 同步到其他电脑
mesh sync

# 单独推送
mesh push

# 单独拉取
mesh pull
```

## 四、与现有 Memory 兼容

### 4.1 导入 Markdown

```bash
# 导入现有 memory
mesh import /path/to/memory.md
```

### 4.2 导出 Markdown

```bash
# 导出为 Markdown
mesh export > memory.md

# 指定输出文件
mesh export --output memory.md
```

## 五、常用场景

### 5.1 AI 对话后保存

```bash
# 与 Claude 聊完产品决策后
mesh collect --source "claude" --content "决策：xxx" --tag "产品,决策"
```

### 5.2 开始新对话前查询

```bash
# 查询相关上下文
mesh query "定价策略"

# 注入到新对话
mesh inject --topic "定价策略" --format prompt
```

### 5.3 多电脑协作

```bash
# 电脑 A 工作
mesh collect --source "claude" --content "完成 xxx 功能" --tag "开发"

# 电脑 B 同步
mesh sync

# 电脑 B 查看进展
mesh query "xxx 功能" --limit 5
```

## 六、Agent 协作

### 6.1 Agent 注册

```bash
# Cursor 首次启动时注册
mesh agent register \
  --id "cursor_main_shawn" \
  --name "Cursor Main" \
  --type "cursor" \
  --device "shawn"
```

### 6.2 Agent 查询上下文

```bash
# Agent 开始任务前
CONTEXT=$(mesh query "定价策略" --format json)
```

### 6.3 Agent 上报进展

```bash
# Agent 完成任务后
mesh collect --source "cursor_main_shawn" --content "完成 xxx" --tag "任务,完成"
```

---

## 七、故障排除

### 7.1 同步冲突

```bash
# 如果遇到冲突
mesh pull --force    # 强制拉取
mesh push --force    # 强制推送
```

### 7.2 数据库损坏

```bash
# 删除数据库重新初始化
rm ~/.mesh/mesh.db
mesh init
```

### 7.3 查看日志

```bash
# 启用详细输出
mesh --verbose collect --source "claude" --content "xxx"
```
