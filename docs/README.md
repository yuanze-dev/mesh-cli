## 📖 项目概览

**Mesh** 是一个极简的 CLI 工具，用于：
- 多电脑之间的记忆同步
- 多 AI 软件之间的上下文共享
- 人 + 多 Agent 的协作

**核心特点**：
- ❌ 无用户注册
- ❌ 无服务器架构
- ✅ 极简优先
- ✅ 单文件可执行
- ✅ 兼容现有 Markdown Memory
